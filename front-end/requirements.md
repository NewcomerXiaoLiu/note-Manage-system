# 个人笔记管理系统 - 功能需求文档

## 1. 整体需求
实现一套轻量化个人笔记管理平台，支持笔记分类分组、笔记编辑管理、数据统计看板，满足个人日常笔记记录与整理需求。

## 2. 页面与菜单需求
### 2.1 菜单规则
- 系统菜单为**固定菜单**，前端写死，不动态生成
- 包含两大核心模块：工作台、笔记管理体系

### 2.2 页面结构
1. **工作台（首页）**
2. **笔记分组管理**
3. **笔记列表管理**

## 3. 详细功能需求

### 3.1 工作台模块
- 展示笔记总数统计
- 展示分组总数统计
- 展示近期新增笔记数据
- 数据由后端接口统一返回

### 3.2 笔记分组模块
- 新增笔记分组
- 编辑分组名称、排序
- 删除分组
- 分组作为笔记分类筛选条件

### 3.3 笔记模块
- 创建笔记（绑定对应分组）
- 编辑笔记标题、内容、所属分组
- 删除笔记
- 根据分组筛选笔记列表
- 查询笔记详情

## 4. 业务约束
1. 所有笔记**必须归属一个分组**
2. 分组为系统核心分类维度，支撑整个笔记筛选体系
3. 前端只负责展示与交互，所有数据来源于后端接口
4. 所有数据持久化存储在 Cloudflare D1

## 5. 接口需求
- 工作台统计接口
- 笔记分组 CRUD 接口
- 笔记 CRUD、筛选、详情接口
